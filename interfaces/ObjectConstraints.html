Optional depthFactor

depthFactor?: number
  • Defined in src/arbitrary/_internals/helpers/QualifiedObjectConstraints.ts:24

Limit the depth of the object by increasing the probability to generate simple values (defined via values) as we go deeper in the object.

Example of values: 0.1 (small impact as depth increases), 0.5, 1 (huge impact as depth increases).

Optional values

values?: Arbitrary<unknown>[]
  • Defined in src/arbitrary/_internals/helpers/QualifiedObjectConstraints.ts:70

Arbitrary for values

Default for values are:

  • boolean,
  • integer,
  • double,
  • string
  • constants among:
  • null,
  • undefined,
  • Number.NaN,
  • +0,
  • -0,
  • Number.EPSILON,
  • Number.MIN_VALUE,
  • Number.MAX_VALUE,
  • Number.MIN_SAFE_INTEGER,
