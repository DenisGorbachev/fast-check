Class ArbitraryWithShrink<T>

Abstract class able to generate and shrink values on type T

It can shrink a value that it has not produced through generate (contrary to Arbitrary). In the case of classical Arbitrary there is no shrink or shrinkableFor methods directly on the Arbitrary, the users have to call shrink on the instance of Shrinkable produced by generate.
